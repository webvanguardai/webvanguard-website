## Lo que propongo: dos modelos, escoge uno

### Opción A — Separación clara (más rápida de implementar)

**True Love Creative** = diseño premium, branding, identidad visual, proyectos creativos de autor. Clientes: marcas que buscan diferenciación estética, proyectos personales, arte, cultura.

**Web Vanguard** = agencia de negocio digital. Webs, SEO, conversión, IA. Clientes: negocios Dubai que quieren más clientes, no solo una web bonita.

**Sistema de derivación:** 
- Cuando Web Vanguard cierra un cliente que también necesita branding → referido a TLC (Javi factura dos veces)
- Cuando TLC recibe un cliente que necesita web → referido a Web Vanguard
- Sin comisiones artificiales — son la misma persona, pero dos propuestas distintas dan más credibilidad a cada servicio

**Ventaja:** Inmediato. No requiere cambiar nada en ninguna de las dos webs.

---

### Opción B — Bundle premium (más potente, más trabajo)

Crear una **oferta conjunta** bajo una de las dos marcas (propongo Web Vanguard como marca paraguas para Dubai):

**"The Full Stack" — AED 12,000–18,000:**
- Identidad visual completa (logo, paleta, tipografía, guidelines) → True Love Creative
- Website premium (Next.js, animaciones, editorial) → Web Vanguard  
- SEO foundation → Web Vanguard
- WhatsApp + AI agent básico → Web Vanguard
- Entregado en 3 semanas

**Por qué funciona:** Ninguna agencia en Dubai ofrece esto completo en 3 semanas. Las grandes cobran AED 40-80K y tardan meses. Los freelancers no tienen el stack completo. Nosotros sí.

**Target:** Startups y negocios premium que están lanzando desde cero y quieren hacerlo bien de entrada.

## El diferenciador que nadie más tiene: el AI Agent

Esto es lo que más me interesa desarrollar como CEO de Web Vanguard, porque nadie en Dubai lo ofrece a precio accesible:

**"Your website + your AI assistant"**

Un agente de WhatsApp entrenado en el negocio del cliente:
- Responde FAQs 24/7 ("¿Tenéis mesa para 4 el viernes?")
- Califica leads ("¿Cuál es tu presupuesto?")
- Captura datos (nombre, teléfono → CRM automático)
- Hace seguimiento post-visita

**Precio mercado actual:** AED 15,000-40,000 (agencias tradicionales)
**Nuestro precio:** AED 1,500 setup + AED 300/mes mantenimiento

**Por qué podemos:** Tenemos OpenClaw + Claude. El coste marginal es mínimo. El valor percibido es enorme.

**Cómo lo presentamos:** No como "chatbot" (suena genérico). Como "un empleado digital que trabaja 24/7 por AED 300/mes."
