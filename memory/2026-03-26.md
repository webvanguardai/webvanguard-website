**Built new demo: Saffron House** (Boutique Hotel — Al Fahidi, Old Dubai)

Design decisions to break the mold:
- Entry screen = full-height atmospheric scene, NO hero image (pure CSS gradients + Islamic geometric SVG)
- Layout: room presentation as FULL-WIDTH stacked alternating panels (image ↔ info)
- NO generic hero → services → contact structure
