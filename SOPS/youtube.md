# SOP YouTube v1 - WebVanguard

## Propósito
Estandarizar la creación de guiones para el canal de YouTube de WebVanguard, maximizando CTR y retención.

## Template de Guion (5-8 min)

### 1. Hook (0-15s)
- **Fórmula:** Problema/Curiosidad/Contraste + Promesa
- Ejemplo: "¿Tu web premium está perdiendo ventas? Te muestro cómo rediseñarla para convertir."

### 2. Intro (15-45s)
- Presentación rápida: "Soy [nombre], de WebVanguard, y hoy..."
- Contextualización del problema.
- Lo que el viewer aprenderá (3 puntos max).

### 3. Desarrollo (3-5 min)
- Dividido en segmentos claros (usa pantallas/key moments).
- Cada segmento: explicación breve + demo o ejemplo visual.
- Ritmo: corto, directo, sin relleno.

### 4. CTA (últimos 30s)
- Pregunta: "¿Qué opinas? Comenta abajo."
- Like/Suscribirse recordatorio sutil.
- Enlace a recurso (landing, plantilla, contacto).

## Checklist CTR (Antes de publicar)
- [ ] Título: inventivo + número/palabra poderosa + keywords (ej. "3 errores que matan tu web premium").
- [ ] Miniatura: fondo oscuro, texto bold <5 palabras, contraste alto, rostro o elemento gráfico llamativo.
- [ ] Primeros 15s: gancho visual/audio que no se puede saltar.
- [ ] Tags: hasta 15, mezcla de keywords específicas y genéricas.
- [ ] Descripción: primeras 2 líneas con keywords + enlaces; luego guion o bullet points.

## Checklist Retención
- [ ] Ritmo: cada 30-60s hay un punto de valor (ej. "tip", "ejemplo", "cambio de pantalla").
- [ ] Gráficos/pantallas clave cada 1-2 min (evitar talking head monótono).
- [ ] Transiciones suaves; eliminar pausas largas.
- [ ] Final con resumen de 3 takeaways y llamado a acción claro.

## Ejemplo Guion - Video: "WebVanguard: El diseño que vende más"

**Hook (0-10s):**
Voz: "Imagina una web tan premium que tus clientes sienten que ya han comprado antes de llamarte. Te muestro cómo lo hacemos."

**Intro (10-40s):**
"Somos WebVanguard, estudio con sede en Abu Dhabi. Hoy desglosamos la anatomía de una web que convierte visitas en leads de lujo."

**Desarrollo:**
- Segmento 1: Hero que autoriza (ejemplo Dubai clinic, min 1:00)
- Segmento 2: Navegación que guía (ejemplo fintech, min 2:30)
- Segmento 3: SEO IA-friendly (ejemplo real estate, min 4:00)
- Segmento 4: CTA psicológico (ejemplo hospitality, min 5:30)

**CTA:**
"Si quieres que tu web hable de tu valor, visita webvanguard-website.vercel.app y descarga el brief template. Suscríbete para más guiones de venta."

## Herramientas sugeridas
- Guión: Notion/Google Docs
- Gráficos: Figma/Canva (tamaño 1280x720)
- Edición: DaVinci Resolve / Premiere Pro
- Miniaturas: Photoshop o Canva

## Métricas a revisar post-publicación
- CTR (impresiones a visitas) > 5%
- Retención a 30s > 70%
- Retención a 2min > 40%
- Engagement (likes/comentarios) > 2% de vistas
