Javi tiene tres marcas personales distintas:

1. **javibeat.com** — DJ internacional, 20+ años, Dubai. Beach clubs, corporate, weddings.
2. **estrela.photo** — Fotografía. Estética propia, proyecto personal.

4. **webvanguard.co** — Agencia de webs para negocios en Dubai. Conversión, SEO, rapidez.

El problema: **cuatro marcas, ninguna hablando con las otras.** El potencial cruzado está sin explotar.

## La Oportunidad Real

### Clientes que necesitan a ambas marcas

Un restaurante de lujo en DIFC necesita:
- Un website que convierte (→ Web Vanguard)
- Una identidad visual coherente, logo, paleta, fotografía (→ True Love Creative + estrela.photo)

Ahora mismo ese cliente tiene que contratar tres sitios distintos y coordinar todo solo. **¿Y si lo ofrecemos como un paquete?**

Un DJ residente en un hotel de Dubai necesita:
- Una web personal que impresione (→ Web Vanguard o TLC)
- Fotos profesionales para el EPK (→ estrela.photo)
- Quizás un set mix bien producido y editado

Ahora mismo Javi lo haría gratis para un amigo. **¿Por qué no cobrarlo como servicio?**

## Propuestas Concretas

### Opción 1 — Paquete "Full Brand" (la más sencilla de vender)

**Web Vanguard** hace el website (AED 5,000-8,000).
**True Love Creative** hace el branding (logo, colores, tipografía) (AED 3,000-6,000).
**estrela.photo** hace las fotos del negocio para el web (AED 1,500-3,000).

Total paquete: **AED 9,500–17,000** — facturado por una sola entidad.
Vs. cliente contratando cada cosa por separado: más caras, más fricción, peor resultado porque nadie coordina.

**¿Por qué funciona?** Porque el cliente solo tiene una persona de contacto (Javi / Web Vanguard) y obtiene un resultado coherente donde web, marca e imágenes hablan el mismo idioma visual.

### Opción 2 — Referidos con comisión

Más simple de arrancar:
- Si TLC tiene un cliente que necesita web → manda a Web Vanguard → 15% de comisión
- Si Web Vanguard tiene un cliente que necesita branding → manda a TLC → 15% de comisión
- Si alguien necesita fotos → estrela.photo → sin comisión, es Javi mismo

Cero administración extra. Solo registrar las referencias.

### Opción 3 — El ecosistema Javi Beat

Javi como DJ tiene acceso a hoteles, beach clubs, marcas de lujo en Dubai. Esos son exactamente los clientes que Web Vanguard busca.

¿Y si los eventos de Javi como DJ son también oportunidades de negocio para Web Vanguard? Dejando una tarjeta o QR en el evento, o directamente hablando con el venue manager. Un DJ que lleva años en los mejores locales de Dubai tiene una red que ninguna agencia puede comprar.
